# Cetris

Clone of Tetris game, written in C++, using SFML 2.5 library.

### How to compile Cetris

**On Linux**

Requirements: SFML 2.5

```bash
mkdir build
cd build
cmake ..
make
```

### Steering

* **W, S, A, D** -> Move the tetromino: Up, Down, Left, Right
* **X** -> Drop the tetromino
* **ESC** -> Quit the game
* **R** -> Restart game in "Game Over" screen
